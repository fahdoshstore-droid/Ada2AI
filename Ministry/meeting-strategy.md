## 2. إستراتيجية الاجتماع

### المرحلة 1: ما قبل الاجتماع (1-2 أسبوع)
- [ ] إرسال Email تعريفي رسمي
- [ ] إرسال Pitch Deck PDF
- [ ] متابعة عبر X (تفاعل مع محتوى الوزارة)
- [ ] تسجيل في برنامج انطلاقة إن أمكن
- [ ] تجهيز Demo حي للمنصة

### المرحلة 2: الاجتماع (30-45 دقيقة)
**الجدول المقترح:**
| الوقت | المحتوى | الهدف |
|-------|---------|-------|
| 0-5 د | مقدمة وتمهيد | بناء الثقة |
| 5-15 د | المشكلة + السوق | إثبات الحاجة |
| 15-25 د | الحل + Demo حي | إثبات القدرة |
| 25-35 د | التوافق مع رؤية 2030 | إثبات القيمة |
| 35-45 د | النقاش + Next Steps | الاتفاق |

### المرحلة 3: بعد الاجتماع (خلال 48 ساعة)
- [ ] إرسال ملخص الاجتماع + Action Items
- [ ] إرسال عرض مخصص حسب ملاحظاتهم
- [ ] جدولة اجتماع متابعة خلال أسبوعين

## 3. أسئلة متوقعة + إجابات

| السؤال | الإجابة المقترحة |
|--------|-----------------|
| ما الذي يميزكم عن Wyscout؟ | نحن الوحيدون بالعربية، مصممون للسوق السعودي، ومعايير SAFF + FIFA |
| ما هي دقة تحليلكم؟ | 89% دقة عامة، 96% في object detection، نتحسن باستمرار مع بيانات سعودية |
| ما تكلفة المشروع التجريبي؟ | سنقدمه مجاناً للوزارة — 3 أشهر كاملة كشراكة |
| كيف تضمنون خصوصية البيانات؟ | تشفير end-to-end، استضافة سعودية، توافق مع PDPL |
| هل لديكم خبرة في القطاع الرياضي؟ | الفريق يجمع بين الخبرة التقنية والمعرفة الرياضية المحلية |
| ما المدة للإطلاق الكامل؟ | 3 أشهر للتجريبي، 6-9 أشهر للتوسع الوطني |
| هل المنصة جاهزة الآن؟ | نعم — MVP يعمل، نحتاج فقط التكامل مع أنظمة الوزارة |

### قاعدة المتابعة
- **القاعدة الذهبية**: لا تنتظر أكثر من 10 أيام بدون تواصل
- **التوازن**: 2 email : 1 مكالمة : 1 X interaction
- **القيمة**: كل تواصل يجب يضيف قيمة (خبر، إحصائية، feature جديد)
